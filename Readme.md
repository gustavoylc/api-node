# API developed with Nodejs for educational purposes
 
 ## Getting Started

```Javascript
npm install
npm run dev
```

 This app was built to show the basic use of Express, using:
 environment variables
 sequelize ORM (sequelize-cli)
 nodemon
 docker with postgresql database
 libraries:
  JOI
  @hapi/dom
  cors
  pg-hstore
  pg
